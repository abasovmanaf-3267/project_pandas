# project_pandas

Pandas Core Functionalities Applied
In this project, I leveraged the full power of Pandas to transform raw data into actionable insights:

Data Cleaning & Wrangling:

Systematically handled NaN values using fillna() and dropna() techniques.

Identified and removed duplicate records to ensure data integrity.

Standardized data types (e.g., converting objects to string or int for memory optimization).

Data Transformation:

Applied merge() to combine multiple datasets (orders, products, departments, etc.) for a holistic view.

Used pivot_table() to generate cross-tabulations for time-based activity analysis.

Advanced Data Analysis:

Performed complex filtering using boolean indexing.

Utilized groupby() and agg() for calculating descriptive statistics (min, max, mean).

Computed key business metrics, such as reorder rates and customer purchasing frequency.

Anomaly Detection:

Investigated data limitations and system constraints (e.g., mapping add_to_cart_order inconsistencies).
